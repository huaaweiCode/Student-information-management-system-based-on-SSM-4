# 基于SSM的学生信息管理系统4

#### Description
学生信息管理系统拥有两种角色，分别为学生和管理员，可以作为初学者使用学习。

**学生**：个人信息查看和修改、查看班级信息

**管理员**：管理员对学生信息、老师信息、学院年级信息的添加、修改、删除、多条件查询等功能
